 - Move the folder anywhere you please depending on the version you want to use.
 - Download and install Forge
   - 1.7: http://files.minecraftforge.net/maven/net/minecraftforge/forge/1.7.10-10.13.4.1558-1.7.10/forge-1.7.10-10.13.4.1558-1.7.10-installer-win.exe
   - 1.8: http://files.minecraftforge.net/maven/net/minecraftforge/forge/1.8-11.14.4.1563/forge-1.8-11.14.4.1563-installer-win.exe
 - Select the Forge version in your Minecraft launcher profile
 - Launch Minecraft and wait until you are in the main menu
 - Run Vape Launcher and wait a couple of seconds
 - Join your favourite server
 - Press ` to open the GUI (or right-shift for non-English keyboards)


Keybindings:
-----------------------------

 - There are no default keybinds. To open the GUI on Vape 2.0 and above version press Grave(`) for english keyboards and right-shift for everyone else
 - To bind a key, hold left shift and click the button corresponding to the module, it will prompt you for a key. Alternatively you can use the bind command "bind hackname keyname"
 - To unbind, shift click the button corresponding to the module, it will prompt you for a key, just click the box again. Or "bind hackname none"

Everything

Syncing / saving settings
